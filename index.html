Text-Operations-Tool

Welcome to the Text-Operations-Tool! This tool allows you to perform various text operations such as counting words, reversing text, and converting text to uppercase or lowercase.

Available Operations:

  • Count Words: Count the number of words in a given text.
  • Reverse Text: Reverse the order of characters in the text.
  • Uppercase: Convert all characters in the text to uppercase.
  • Lowercase: Convert all characters in the text to lowercase.

How to Use:

  1. Select the desired operation from the list above.
  2. Input your text in the provided text area.
  3. Click the "Submit" button to perform the operation.
  4. View the result displayed below the input area.

Example Usage:

If you input the text "Hello World" and select "Count Words", the result will be "2".

If you input the text "Hello World" and select "Reverse Text", the result will be "dlroW olleH".

If you input the text "Hello World" and select "Uppercase", the result will be "HELLO WORLD".

If you input the text "Hello World" and select "Lowercase", the result will be "hello world".
